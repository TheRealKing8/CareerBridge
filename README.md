# CareerBridge

**Tagline:** Connecting students, graduates, and employers through opportunities.

CareerBridge is a platform that connects students, graduates, and employers through opportunities — internships, attachments, graduate trainee programs, and full-time jobs.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Tech Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS 4
- Prisma (database) — _coming in next step_
- NextAuth — _coming_
- React Hook Form + Zod — _coming_

## Roles

- **STUDENT** — find opportunities, build profile, upload CV, track applications, AI career guidance
- **EMPLOYER** — post jobs, review applicants, manage recruitment
- **ADMIN** — moderate employers, manage jobs, view analytics
